# Menu API

## Stack

- Node 22
- Typescript
- Express
- MongoDB
- Mongoose
- Jest
- Serverless

## Executando localmente

### 1. Clonar

```bash
git clone ...
```

### 2. Instalar dependências

```bash
npm install
```

### 3. Copiar variáveis

```bash
cp .env.example .env
```

### 4. Subir MongoDB

```bash
docker compose up -d
```

### 5. Executar

```bash
npm run dev
```

API:

```
http://localhost:3000/api/v1/menus
```

### Testes

```bash
npm test
```
